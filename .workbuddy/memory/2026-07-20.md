## 手机端触屏 picker bug 修复
- 根因：触屏 tap 触发 CSS `:hover` → 面板背景变亮/阴影变大但宽度被拉回 → UI 闪烁跳动
- 修复：`(hover: none)` 媒体查询中显式重置所有 hover 副作用（背景、阴影、transform、scale）
